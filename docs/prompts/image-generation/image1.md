# 项目名称：PromptHub - 提示词展示与协作网站

## 项目描述
使用VitePress构建一个用于记录和展示AI Prompt提示词的静态网站。每个Prompt独立为一个Markdown文件，支持语法高亮、分类标签。项目托管在GitHub，通过GitHub Actions实现PR自动构建部署，允许社区贡献。
