## Description

Generate original short stories with rich characters, compelling plots, and vivid settings. Supports genre specification and style customization.

## Prompt

```
You are a published fiction author. Write an original short story based on the following parameters.

Genre: [fantasy / sci-fi / mystery / romance / horror / literary / historical fiction]
Setting: [describe the world, time period, and atmosphere]
Protagonist: [name, age, defining trait, motivation]
Central conflict: [what is at stake? what is the obstacle?]
Tone: [dark / lighthearted / suspenseful / melancholic / hopeful]
Approximate length: [500 / 1000 / 2000 words]

Story structure:
1. Opening — establish setting and character in the first 2-3 sentences
2. Rising action — introduce the conflict naturally
3. Climax — the turning point
4. Resolution — satisfying conclusion (not necessarily happy)

Style notes:
- Show, don't tell — use sensory details and action instead of exposition
- Include at least one moment of genuine surprise or emotional impact
- Dialogue should reveal character, not just advance plot
- End with a line that resonates

Write the complete story.
```

## Tips

- The more specific you are about the protagonist's internal conflict, the more compelling the story.
- For serialized stories, end with a mild cliffhanger and note it's "Part 1."
- Combine with the [Character Designer](/prompts/creative/character-designer) prompt to build characters first.
