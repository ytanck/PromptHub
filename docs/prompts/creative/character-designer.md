## Description

Create detailed, three-dimensional characters for stories, RPGs, or worldbuilding. Covers personality, backstory, appearance, and character arcs.

## Prompt

```
You are a character designer for fiction and games. Create a detailed character profile based on the following parameters.

Character role: [protagonist / antagonist / sidekick / mentor / love interest / villain]
Genre/Setting: [fantasy / modern / sci-fi / historical / etc.]
Name: [leave blank for AI to generate, or provide a name]

Provide a complete profile covering:

## Basic Information
- Full name and any nicknames
- Age, gender, species (if applicable)

## Appearance
- Height, build, distinguishing features
- Typical clothing/style
- How others perceive them at first glance

## Personality
- Myers-Briggs or Enneagram type (approximate)
- Core values and beliefs
- Strengths (2-3) and flaws (2-3)
- Communication style and mannerisms
- What they fear most / what they desire most

## Backstory
- Key life events that shaped them (3-5 bullet points)
- Current occupation and living situation
- Relationships: family, friends, rivals

## Character Arc
- Where they start vs. where they end
- The lesson they need to learn
- Internal conflict that drives their decisions

## Dialogue Sample
- Write 3-4 lines of dialogue that showcase their voice
- Include a note on speech patterns (formal, slang, terse, verbose)

Make the character feel real — avoid clichés. Give them contradictions and complexity.
```

## Tips

- For RPG characters, add an "Abilities & Skills" section with game-specific attributes.
- Pair this with the [Story Generator](/prompts/creative/story-generator) to place characters in a narrative.
- The more specific the genre and setting, the more coherent the character will be within their world.
