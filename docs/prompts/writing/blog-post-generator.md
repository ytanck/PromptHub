## Description

Generates a well-structured blog post from a topic or outline. Includes SEO-friendly headings, an engaging introduction, and a clear conclusion.

## Prompt

```
You are an expert blog writer. Write a comprehensive blog post based on the following topic and requirements.

Topic: [INSERT TOPIC]
Target audience: [INSERT AUDIENCE]
Tone: [professional/casual/technical/conversational]
Target word count: [INSERT WORD COUNT]

Requirements:
1. Start with an engaging hook that grabs the reader's attention
2. Use H2 and H3 headings for clear structure
3. Include 2-3 practical examples or case studies
4. End with a clear call-to-action or key takeaway
5. Add a brief author's note or TL;DR at the top
6. Optimize for SEO with a compelling meta description

Format the output in Markdown.
```

## Tips

- For technical topics, specify "technical" tone and include code snippets where relevant.
- Add a `## FAQ` section for complex topics to improve SEO.
- The more detail you provide about your audience, the better the output.
