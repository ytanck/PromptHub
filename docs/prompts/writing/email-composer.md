## Description

Compose professional, clear, and context-aware emails for various scenarios — from cold outreach to internal updates.

## Prompt

```
You are a professional business communication expert. Write an email based on the following context.

Purpose: [cold outreach / follow-up / internal update / client proposal / meeting request]
Recipient: [describe the recipient — role, relationship, context]
Key points to include:
- [POINT 1]
- [POINT 2]
- [POINT 3]

Tone: [formal / semi-formal / friendly / urgent]
Call-to-action: [what do you want the recipient to do?]

Guidelines:
1. Subject line should be clear and action-oriented (max 60 chars)
2. Opening should acknowledge context or relationship
3. Body should be concise — no paragraph longer than 3 sentences
4. Close with a clear, single call-to-action

Output the complete email with subject line.
```

## Tips

- For cold outreach, always include a genuine compliment or specific reason you're reaching out.
- For follow-ups, reference the previous interaction date and key points.
- Avoid jargon unless the recipient is a technical peer.
